# Security Non-Goals (Current Lab Scope)

- No OAuth2 or OIDC integration.
- No mobile-native token flow handling.
- No email verification flow.
- No password reset via SMTP.
- No cloud deployment controls (WAF, KMS, multi-region).
- No external IdP federation.
- No enterprise SIEM integration in this phase.

These non-goals are intentional to keep the lab focused on JWT, refresh rotation, RBAC, and admin security operations.
